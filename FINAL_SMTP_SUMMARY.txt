🔍 HOW SMTP VERIFICATION WORKS
═══════════════════════════════════════════════════════════════════════════════

Process:
  1. Connect to mail server (SMTP port 25)
  2. Send EHLO command
  3. Send MAIL FROM command
  4. Send RCPT TO command with email address
  5. Check response:
     ✅ 250 = Mailbox exists (VERIFIED)
     ❌ 550/551/552 = Mailbox doesn't exist (FAILED)
     ⏱️  Timeout = No response (TIMEOUT)

Verification Method: Non-intrusive
  ✓ No test emails sent
  ✓ No spam flags triggered
  ✓ No IP reputation damage
  ✓ Fast (2,499 emails in ~3 minutes)
